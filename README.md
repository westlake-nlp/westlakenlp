# WestlakeNLP

The official homepage of **WestlakeNLP**, a research lab at [Westlake University](https://en.westlake.edu.cn/) led by **Prof. Yue Zhang**.

Founded in September 2018, the lab pursues fundamental and applied research in Natural Language Processing, large language models, and general artificial intelligence — combining linguistic insight with modern machine learning to build language systems that reason, generalize, and assist scientific discovery.

🌐 **Live site:** [westlakenlp.com](https://westlakenlp.com)

## About this repository

This repo contains the source for the lab's website — a static site served via GitHub Pages.

```
.
├── index.html          # Landing page
├── team.html           # Members and alumni
├── publications.html   # Selected publications
├── work.html           # Research directions
├── teaching.html       # Courses and resources
└── assets/             # Styles, scripts, and images
```

## Local development

No build step is required. Clone the repo and open `index.html` in a browser, or serve the directory with any static server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Contributing

Lab members can update content by editing the relevant HTML file and opening a pull request. Please keep image assets optimized and place them under `assets/images/`.
